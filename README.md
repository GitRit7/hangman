HANGMAN 0_0

Get 6 chances to guess the word.

word to be guessed is stored in the editible word.txt.
length of word to be guessed is given.
type word, if you are correct you win.
if not, you will be given the letters that were right at the right positions.
checks if you gave a guess with incorrect length.

how to run:

clone repo
write word to be guessed in word.txt
open in any IDE then run
or open in terminal:-
    (currect directory should be project folder location)

    javac Main.java
    java Main

    (for this, java path should be created already)
